I wanted to host my own website to learn more about web hosting and server management in general. Additionally, it is a great way to present myself and share some information about me with others.
A domain can be acquired quite cheaply. But hosting a webpage on a server is more expensive. Starting with this, it opens up a lot of possibilities to host other services or to extend it to my own cloud server. {% endblock %} {% block description %}
